MerchantSafe Unipay PHP SDK

MerchantSafe Unipay (MSU) is an online payment solution developed by Asseco SEE Turkey. This SDK is not an official SDK. This library written since Asseco does not provide any PHP SDK.

Installation

You need Composer to install MerchantSafe Unipay PHP SDK.

This library requires PHP 7.0 or later. 


composer require mkorkmaz/msu-php-sdk
